# Quant Tool

面向 A 股研究场景的本地量化决策辅助工具。当前版本是 macOS 东方财富伴随侧栏：只读识别当前股票，并调用本机量化服务展示真实历史研究覆盖状态。

## 当前进度

- P0：东方财富进程、股票代码、窗口位置识别与实时跟随已经实现；
- P1：Tauri + Vue 桌面端、FastAPI 本地服务、模拟诊断评分卡和服务异常恢复已经实现；
- P2 研究数据层：真实日线、指数、Point-in-Time 财务数据和 15 个因子已经接入；
- P3 真实研究验证：Walk-Forward、三种排序模型、Top 20 组合和 A 股约束回测已经跑通；
- P4 当前日频信号已接通：最新完整日线生成横截面排名，并与历史样本外成绩分开展示；
- P4 技术形态已接通：真实 K 线叠加 MA5/MA20/MA60、每日成交量、VMA5/VMA20、MACD，并自动描绘双底、双顶、突破、跌破、趋势通道和箱体边界；
- 侧栏会区分“当前 Top 20”“研究池内未入选”和“当前研究池未覆盖”，未覆盖股票不会生成个股排名或上涨概率；
- 当前本地信号使用截至 `2026-07-23` 收盘的数据，训练标签截至 `2026-07-09`；
- `002463` 沪电股份与当前股票 `001309` 德明利已纳入 30 股试运行池，本次分别排名第 5、第 6；
- 设置面板提供“更新当前日频数据”，可在应用内完成取数、训练和信号刷新；
- 当前模型因数据准入门禁不得成为默认模型，也不提供自动交易。

## 本地开发

需要：

- Node.js `24.18.0`；
- pnpm `8.15.4`；
- Rust `1.97.1`；
- [uv](https://docs.astral.sh/uv/)。

```bash
nvm use
pnpm install
pnpm dev
```

`pnpm dev` 会自动：

1. 安装并锁定 Python 3.12 本地服务环境；
2. 启动 Vue 开发服务器；
3. 启动 Tauri 桌面应用；
4. 由 Rust 在随机端口启动只监听 `127.0.0.1` 的 FastAPI 服务。

检查全部代码：

```bash
pnpm check
```

更新 P2 真实研究数据并重新运行 P3：

```bash
pnpm research:p2
```

刷新到数据源最新完整交易日并生成当前日频信号：

```bash
pnpm research:daily
```

同一任务也可以在侧栏设置中点击“更新当前日频数据”执行。

只使用已经下载的 P2 工件重跑 P3：

```bash
pnpm research:p3:real
```

生成 OpenAPI 和 TypeScript 共享协议：

```bash
pnpm contracts:generate
```

构建 macOS App：

```bash
pnpm build
```

当前构建仍从仓库内启动 Python 环境，只用于本机开发和验证；到 P5 再将服务打包为可分发 sidecar。

## 安全与合规边界

- Accessibility API 仅用于读取当前股票和窗口位置；
- 不读取账号、密码、验证码、资金或真实持仓；
- 不注入、Hook 或模拟东方财富交易操作；
- 本地 API 仅监听回环地址，并使用每次启动生成的会话令牌；
- 当前排名是最新完整日线信号，不是分钟级盘中行情；模型未通过准入，不构成证券投资建议。
- 技术形态由规则自动识别，拐点确认存在滞后，只作为图形研究辅助。
