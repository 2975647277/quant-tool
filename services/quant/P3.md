# P3：回测与排序模型

## 当前实现

P3 已建立可替换数据输入的研究闭环：

```text
RankingDataset
      ↓
Walk-Forward（10个交易日隔离）
      ↓
规则评分 / 线性因子 / LightGBM Ranker
      ↓
Rank IC、ICIR、Top组超额收益
      ↓
Top 20 等权模拟组合
      ↓
A股约束与成本回测
```

当前使用确定性研究样本完成工程验收。P2 真实数据层尚未完成，因此这些数值不代表真实策略收益，任何模拟模型都不能注册为默认模型。

## 排序模型

实现了三条模型基线：

1. `rule-score-v1`：固定权重的规则评分；
2. `linear-factor-ridge-v1`：标准化因子加 Ridge 线性模型；
3. `lightgbm-lambdarank-v1`：按交易日分组、五档相关性标签的 LambdaRank。

第一批研究特征暂为：

- 动量；
- 质量；
- 价值；
- 低波动；
- 流动性；
- 反转。

模型输出会记录：

- 模型版本；
- 数据版本；
- 预测时间；
- 股票代码；
- 预测分数；
- 对应的未来超额收益标签。

## Walk-Forward

当前研究配置：

- 训练集：50 个交易日；
- 训练与验证之间隔离：10 个交易日；
- 验证集：10 个交易日；
- 验证与测试之间隔离：10 个交易日；
- 测试集：10 个交易日；
- 每次向前滚动：10 个交易日。

随机拆分不允许用于时间序列模型验收。

## A股回测约束

信号在交易日收盘后生成，最早在下一交易日开盘执行。引擎覆盖：

- T+1；
- 停牌和零成交量；
- 涨停不可买；
- 跌停不可卖；
- 100 股整手；
- 单日成交量容量；
- 买卖佣金；
- 卖出印花税；
- 双向过户费；
- 买卖滑点。

默认研究成本参数：

| 项目 | 默认值 |
|---|---:|
| 佣金 | 成交额 0.03%，最低 5 元 |
| 卖出印花税 | 成交额 0.05% |
| 过户费 | 成交额 0.001%，双向 |
| 滑点 | 5 bps |
| 最大成交量占比 | 10% |

这些参数均可配置。P2 接入真实历史数据后，还必须按交易日期加载当时有效的费率和交易规则，不能用当前参数覆盖全部历史。

参考：

- [财政部、税务总局关于减半征收证券交易印花税的公告](https://shanghai.chinatax.gov.cn/gate/big5/shanghai.chinatax.gov.cn/tax/zcfw/zcfgk/yhs/202308/t468451.html)
- [股票交易过户费调整说明](https://www.xinhuanet.com/2022-04/28/c_1128605983.htm)
- [LightGBM LGBMRanker](https://lightgbm.readthedocs.io/en/stable/pythonapi/lightgbm.LGBMRanker.html)

## 本地研究接口

```text
GET /v1/research/p3/demo
```

接口需要本地会话令牌，返回：

- 三种模型的 Rank IC、ICIR、Top 组超额收益和最大回撤；
- Walk-Forward 折数和隔离天数；
- 最新 Top 20 等权组合；
- 回测收益、回撤、换手率、成交数和受阻委托数；
- 模型版本、数据版本和预测时间；
- 默认模型准入结果。

当前 ICIR 按每日 Rank IC 的均值除以标准差，并乘以 `√252` 年化。
Top 组平均超额收益使用全部测试日；累计超额收益和对应回撤只取每 10 个交易日一个不重叠样本，避免把重叠的未来 10 日标签重复复利。

也可以直接在仓库根目录运行：

```bash
pnpm research:p3
```

## 默认模型准入

当前工程门槛：

- Rank IC 不低于 0.02；
- ICIR 不低于 0.8；
- Top 组平均超额收益为正；
- 最大回撤不高于 15%；
- 数据必须不是模拟数据。

任何一项不满足都不会成为默认模型。当前研究样本固定触发：

```text
simulated_data_cannot_be_default
```

因此 `defaultModel` 必须为 `null`。
